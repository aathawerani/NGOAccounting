# CLAUDE.md — Auto-Instructions for NGO Accounting Sessions

At the start of every session:
1. Read `TODO.md` to see which tasks remain
2. Read `PROGRESS.md` for current feature status
3. Start on the first unchecked task (`- [ ]`) and work top-to-bottom
4. Mark each task `- [x]` in `TODO.md` as soon as it is complete
5. Update `PROGRESS.md` with changes made in the session
6. At session end add a `### Session YYYY-MM-DD` block to `PROGRESS.md`

Do not ask for permission before starting the next task — just work through the list.
